# 🎓 EduInsight – Student Analytics using Machine Learning  

EduInsight is an AI-powered student analytics system that predicts student performance and provides personalized tutoring suggestions. It uses machine learning models to analyze academic and behavioral data, helping identify at-risk students and improve learning outcomes.

---

## 🚀 Features  

- 📊 Predicts student performance (Pass / Fail)  
- 🤖 Machine Learning-based classification model  
- 📈 Displays prediction confidence score  
- 📘 Provides personalized tutoring advice  
- 🌐 Interactive web application using Streamlit  

---

## 🛠️ Tech Stack  

- Python  
- Pandas, NumPy  
- Scikit-learn  
- Streamlit  
- Joblib  

---

---

## ⚙️ How It Works  

1. User inputs student details such as:
   - Age, Class  
   - Study hours  
   - Attendance  
   - Subject scores  
   - Previous year performance  

2. The trained ML model processes the data and predicts:
   - Pass or Fail  

3. The system outputs:
   - Prediction result  
   - Confidence score  
   - Personalized study recommendations  

---

## ▶️ How to Run  

1. Install required libraries:
```
pip install pandas numpy scikit-learn streamlit joblib
```

2. Run the application:
```
streamlit run app.py
```

3. Open browser and go to:
```
http://localhost:8501
```
